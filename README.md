# Django-doc-tuto

### Virtual environment 
```py
$ python3 -m venv venv
$ source venv/bin/activate
```

### Installer Django
```py
$ pip install django
```

### Créer un project
```py
$ django-admin startproject mysite
```

### Lancer le serveur
```py
$ python manage.py runserver
```

### Créer une application
```py
$ python manage.py startapp myapp
```
